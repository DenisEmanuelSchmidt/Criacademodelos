Visão Geral

Esta API permite gerenciar dados de login salvos no localStorage do navegador. Você pode salvar, consultar, sincronizar e verificar credenciais de login através dos endpoints disponíveis.

Base URL

http://localhost:8000/api

Endpoints Disponíveis

GET /login-data
Retorna todos os dados de login salvos
Resposta: { "success": true, "data": [ { "id": 1, "email": "usuario@exemplo.com", "created_at": "2025-06-26T19:45:00", "last_login": "2025-06-26T19:45:00" } ], "count": 1 }
GET /login-data/{email}
Retorna dados de login por email específico
Resposta: { "success": true, "data": { "id": 1, "email": "usuario@exemplo.com", "created_at": "2025-06-26T19:45:00", "last_login": "2025-06-26T19:45:00" } }
POST /login-data
Salva ou atualiza dados de login
Requisição: { "email": "usuario@exemplo.com", "password": "minhasenha123" } Resposta: { "success": true, "message": "Dados salvos com sucesso", "data": { "id": 1, "email": "usuario@exemplo.com", "created_at": "2025-06-26T19:45:00", "last_login": "2025-06-26T19:45:00" } }
POST /login-data/sync
Sincroniza dados do localStorage com o banco de dados
Requisição: { "users": { "usuario1@exemplo.com": { "password": "senha123", "createdAt": "2025-06-26T19:45:00.000Z", "lastLogin": "2025-06-26T19:45:00.000Z" } } }
POST /login-data/verify
Verifica credenciais de login
Requisição: { "email": "usuario@exemplo.com", "password": "minhasenha123" } Resposta: { "success": true, "message": "Login válido", "data": { "id": 1, "email": "usuario@exemplo.com", "created_at": "2025-06-26T19:45:00", "last_login": "2025-06-26T19:45:00" } }
DELETE /login-data/{email}
Remove dados de login por email
DELETE /login-data/clear
Remove todos os dados de login
